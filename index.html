Element Queries

Simple example

.demo-text.megapony-max-width-360 {
	color: red;
}
.demo-text.megapony-max-width-280 {
	color: blue;
}

This paragraph will be red if it's width is smaller than 360 px and green if it is smaller than 280 px and the paragraph will have the default color for every width above. Important: A reload is needed after resizing. See important notice in README.md

This paragraph will be also red if it's width is smaller than 360 px and green if it is smaller than 280 px. It's width is the jumping point . The width of the paragraph. NOT the screen width. The available width of the element in the context matters. Compare this p-tag with the p-tag top right. The react differently because the are in another context with different space available. You only have to use the classes megapony-max-width-123 or megapony-min-width-456 in your stylesheets. The classes are applied by the framework automatically. Every linked stylesheet with a title attribute value of "megapony" will be parsed. Not with AJAX and regex, just with simple DOM access document.styleSheets.
